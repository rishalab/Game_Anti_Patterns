Ensure Balance Between Game Controls and Functionality

(191 occurrences)

Game control actions involving mouse, keyboard, joystick, and other devices, should be smooth and the action should be optimized as it increase the usability of game [1].

Context

Game control is an integral part of game play. Having effective, smooth and minimized movements of various devices increases the usability of game.

Problem

Rough and faulty game controls can have negative impact on usability factor.

Solution

Provide game with minimum mouse scrolls, keyboard button, and other devices. Optimized game actions along with smooth game controls increases hassle free game play.

Example

In the case of point and click game, there should not be any action related to keyboard or any other device, as it causes unnecessary control movements for player.

Reference
